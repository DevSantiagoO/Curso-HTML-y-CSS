<h1 align="center"> Curso de HTML (Estructura Web)</h1>


## Introducción

Este repositorio ha sido reestructurado para concentrarse **exclusivamente** en el estudio y la documentación del lenguaje **HTML (HyperText Markup Language)**.

El objetivo es pasar de tener conocimientos básicos a dominar la **estructura semántica y la accesibilidad** del contenido web, sentando una base sólida para el desarrollo de cualquier proyecto.

## Ruta de Aprendizaje y Referencia Principal

La ruta de estudio seguirá la estructura temática y el contenido proporcionado por **Lenguaje HTML** (`https://lenguajehtml.com/html/`), asegurando una cobertura completa de los estándares modernos de HTML.

Nos centraremos en los siguientes pilares clave, que reflejan la organización de la guía:

### 1. Fundamentos y Estructura Esencial
* **Sintaxis y Etiquetas:** Cómo funciona el marcado HTML, sus atributos y la estructura básica.
* **Texto y Enlaces:** Manejo de párrafos, encabezados, listas y la gestión de hipervínculos (`<a>`).

### 2. Contenido Multimedia y Tablas
* **Imágenes y Figuras:** Uso avanzado de la etiqueta `<img>` y el manejo de contenido visual.
* **Elementos Multimedia:** Inclusión y control de video (`<video>`) y audio (`<audio>`) en la web.
* **Tablas de Datos:** Estructuración de información compleja con las etiquetas `<table>`, `<thead>`, `<tbody>`, etc.

### 3. Formularios y Entrada de Datos (La Clave para el Backend)
* **Estructura de Formularios:** Creación de formularios con `<form>`.
* **Tipos de Inputs:** Uso de todos los tipos de entrada (`text`, `email`, `password`, `date`, `file`, etc.).
* **Validación:** Uso de atributos como `required`, `pattern` y `maxlength` para asegurar la calidad de los datos.

### 4. Semántica y Accesibilidad
* **Etiquetas Semánticas:** Uso de `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, y `<footer>`.
* **Accesibilidad (ARIA):** Introducción a las prácticas para que el contenido sea accesible a todos los usuarios.
